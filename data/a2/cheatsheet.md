### A2 German Cheat Sheet: The Cases

Since my previous response was for the A1 level, here is a cheat sheet for the **A2 German language level**. The A2 
level builds on the basics, introducing more complex sentence structures and grammar concepts.

---

## Sentence Structure

At the A2 level, you'll learn to connect ideas and form more complex sentences.

#### 1. Subordinating Conjunctions

These words link a main clause to a subordinate clause. The conjugated verb in the subordinate clause moves to the 
**end**.

| Conjunction | Meaning | Example | Verb Position |
| :--- | :--- | :--- | :--- |
| **dass** | that | *Ich weiß, **dass** du hier **bist**.* | End of clause |
| **weil** | because | *Er lernt Deutsch, **weil** er in Berlin **wohnt**.* | End of clause |
| **wenn** | if/when | *Ich rufe dich an, **wenn** ich Zeit **habe**.* | End of clause |

#### 2. Two-Part Conjunctions

These connect two equal parts of a sentence.

| Conjunction | Meaning | Example |
| :--- | :--- | :--- |
| **entweder ... oder** | either ... or | *Wir können **entweder** ins Kino **oder** ins Theater gehen.* |
| **weder ... noch** | neither ... nor | *Er isst **weder** Fleisch **noch** Fisch.* |

#### 3. Verb Position Rules

| Clause Type | Rule | Example |
| :--- | :--- | :--- |
| **Main clause** | Subject-verb-object pattern | *Ich lerne Deutsch.* |
| **Subordinate clause** | Verb at the end | *..., weil ich Deutsch **lerne**.* |

## Verbs

You'll expand your knowledge of verb tenses and forms.

#### 4. Past Tense (Perfekt)

| Auxiliary Verb | When to Use | Formation | Example |
| :--- | :--- | :--- | :--- |
| **haben** | Most verbs | haben + past participle | *Ich **habe** das Buch **gelesen**.* |
| **sein** | Motion & change of state verbs | sein + past participle | *Er **ist** nach Hause **gegangen**.* |

#### 5. Past Participle Formation

| Verb Type | Formation Rule | Example |
| :--- | :--- | :--- |
| **Regular verbs** | ge- + stem + -t | *machen* → *gemacht* |
| **Irregular verbs** | ge- + stem + -en (with changes) | *trinken* → *getrunken* |
| **Separable verbs** | prefix + ge- + stem + ending | *anrufen* → *angerufen* |

#### 6. Separable Prefixes

| Tense | Rule | Example |
| :--- | :--- | :--- |
| **Present** | Prefix goes to end of sentence | *Ich **rufe** meinen Bruder **an**.* |
| **Past (Perfekt)** | ge- inserted between prefix and stem | *Ich habe meinen Bruder **angerufen**.* |

#### 7. Modal Verbs Overview

| Modal Verb | Meaning | Usage |
| :--- | :--- | :--- |
| **können** | can/to be able to | ability, possibility |
| **wollen** | to want to | desire, intention |
| **müssen** | must/to have to | necessity |
| **dürfen** | may/to be allowed to | permission |
| **sollen** | should/ought to | obligation, recommendation |
| **möchten** | would like to | polite desire |

## Nouns and Cases

You'll master all four German cases.

#### 8. Dative Case

| Article | Nominative → Dative | Example |
| :--- | :--- | :--- |
| **Masculine** | der → dem | *Ich gebe **dem Mann** den Schlüssel.* |
| **Feminine** | die → der | *Ich helfe **der Frau**.* |
| **Neuter** | das → dem | *Das gehört **dem Kind**.* |
| **Plural** | die → den (+n to noun) | *Ich spreche mit **den Kindern**.* |

#### 9. Genitive Case

| Article | Nominative → Genitive | Noun Changes | Example |
| :--- | :--- | :--- | :--- |
| **Masculine** | der → des | +(e)s ending | *Das Auto **des Lehrers** ist rot.* |
| **Feminine** | die → der | no change | *Die Tasche **der Frau**.* |
| **Neuter** | das → des | +(e)s ending | *Die Farbe **des Hauses**.* |
| **Plural** | die → der | no change | *Die Bücher **der Kinder**.* |

#### 10. Two-Way Prepositions with Cases

| Question | Case | Rule | Example |
| :--- | :--- | :--- | :--- |
| **Wo?** (Where?) | **Dative** | Static location | *Der Stuhl steht **im** Wohnzimmer.* |
| **Wohin?** (To where?) | **Accusative** | Movement/direction | *Ich gehe **ins** Wohnzimmer.* |

## Other Essentials

#### 11. Adjective Endings

| Article Type | Case/Gender | Ending | Example |
| :--- | :--- | :--- | :--- |
| **Definite (der/die/das)** | Most cases | -e or -en | *der **rote** Apfel*, *den **roten** Apfel* |
| **Indefinite (ein/eine)** | Varies by case/gender | -er, -e, -es, -en | *ein **roter** Apfel*, *eine **schöne** Frau* |
| **No article** | Varies by case/gender | -er, -e, -es, -en | ***Roter** Wein*, ***kalte** Milch* |

#### 12. Reflexive Verbs

| Subject | Reflexive Pronoun | Example |
| :--- | :--- | :--- |
| **ich** | mich | *Ich **wasche mich**.* |
| **du** | dich | *Du **wäschst dich**.* |
| **er/sie/es** | sich | *Er **wäscht sich**.* |
| **wir** | uns | *Wir **waschen uns**.* |
| **ihr** | euch | *Ihr **wascht euch**.* |
| **sie/Sie** | sich | *Sie **waschen sich**.* |

#### 13. Comparisons

| Form | Formation | Example | Usage |
| :--- | :--- | :--- | :--- |
| **Positive** | Base form | *schön* | *Das Haus ist schön.* |
| **Comparative** | +er (often with umlaut) | *schöner* | *Mein Auto ist **schöner** als deins.* |
| **Superlative** | am +sten or der/die/das +ste | *am schönsten* | *Das ist **das schönste** Haus.* |

## Case Rules Summary

#### 14. The Core Question

To figure out the case, ask a simple question.

| Case | Question | Function |
| :--- | :--- | :--- |
| **Akkusativ** | **Wen oder was?** (Whom or what?) | Direct object |
| **Dativ** | **Wem?** (To whom?) | Indirect object |

#### 15. Accusative & Dative Verbs

Some verbs always trigger a specific case.

| Case | Verbs | Example |
| :--- | :--- | :--- |
| **Akkusativ Verbs** | sehen, kaufen, essen, haben, besuchen, fragen | *Ich **sehe** den Mann.* |
| **Dativ Verbs** | helfen, danken, gefallen, gehören, antworten, gratulieren | *Ich **helfe** dem Kind.* |
| **Two-Object Verbs** | Person (Dativ) + Thing (Akkusativ) | *Ich gebe **dem Kind** **das Buch**.* |

#### 16. Prepositions & Articles

These words lock the noun into a specific case.

| Preposition Type | Prepositions | Case | Article Rule |
| :--- | :--- | :--- | :--- |
| **Always Akkusativ** | **durch, für, gegen, ohne, um** | **Akkusativ** | **der** → **den** (others same) |
| **Always Dativ** | **aus, bei, mit, nach, seit, von, zu** | **Dativ** | **der** → **dem**, **die** → **der**, **das** → **dem** |
| **Two-Way** | **an, auf, hinter, in, neben, über, unter, vor, zwischen** | **Akkusativ** (movement) or **Dativ** (location) | **"wohin?"** → Akkusativ <br> **"wo?"** → Dativ |

#### 17. Dative Endings (Possessive Pronouns)

Remember that possessive pronouns also take an ending in the dative case.

| Noun Gender | Possessive Pronoun Ending | Example |
| :--- | :--- | :--- |
| **Masculine** | **-em** | `zu` **meinem** `Freund` |
| **Feminine** | **-er** | `zu` **meiner** `Mutter` |
| **Neuter** | **-em** | `zu` **seinem** `Kind` |
| **Plural** | **-en** | `zu` **unseren** `Eltern` |
